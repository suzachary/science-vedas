MLA Formatting

General Formatting

  • Paper should be typed/printed on 8.5 by 11 inch paper
  • Text should be double-spaced, in 12-point font, and in a legible font (usually Times New Roman)
  • Leave one space after periods
  • The margins of the document should be 1 inch on all sides
  • Indents should be 1/2 inch from the left margin (using the Tab key)
  • The header should have the page number in the top-right corner, 1/2 inch from the top.
  • Use Italics/Underlining for the titles of longer works and for emphasis
  • Endnotes should be included on a separate page before your Works Cited page, titled "Section Notes"

Citations/Works Cited Page

In-Text Citations

If the author is known, put the author's name and the page number of the source in parentheses. If no author is known, put the name of the source instead. If there are multiple authors, list both names. For electronic sources, no page number is required.

Example: Human beings have been described as "symbol-using animals" (Burke 3).



Works Cited Page

When citing works, list information in this order:

  1. Author
  2. Title of Source
  3. Title of Container
  4. Other Contributors
  5. Version
  6. Number
  7. Publisher
  8. Publication Date
  9. Location
While it is perfectly fine to not have all of the requested information, it is best to include as much information as possible.
